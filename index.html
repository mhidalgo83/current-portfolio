wycoworks

This full-stack application was a contracted project, where I was lead developer, for the University of Kansas. The application was designed with helping those that may have barriers to employment in Wyandotte County. The remote, cross-discipline team aggregated various data sets, including social services, food pantries, education/training services, transit data, and employment opportunities. The front-end is built using React, Redux, Sass, Google Maps API, and the third-party React Google Maps API library to visualize our data. The back-end is built in Node, Express, and MongoDB. No code available, as this is proprietary intellectual property owned by the University of Kansas.
